# WCAG Vurderingsplattform

Dette er en webapplikasjon hvor brukere kan legge til, vurdere og diskutere nettsteder basert på WCAG (tilgjengelighet).

## 🚀 Funksjoner
- Registrering og innlogging med Firebase Auth
- Legge til nettsteder med vurdering
- Stemmesystem (liker/liker ikke)
- Kommentarer på nettsteder
- Rapportering av innhold
- Adminpanel for moderering
- Søk, sortering og paginering

## 🛠️ Teknologi
- JavaScript (vanilla)
- Firebase (Auth + Firestore)
- HTML / CSS

## 🔥 Firebase
Prosjektet bruker Firebase til:
- Autentisering av brukere
- Lagring av data i Firestore
- Sanntids oppdatering av innhold

## 👤 Roller
- Vanlige brukere kan legge til og kommentere nettsteder
- Admin kan slette innhold og håndtere rapporter
